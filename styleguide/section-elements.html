Secondary
Gradient

2.2 Buttons

DESCRIPTION:

Styles for button elements

EXAMPLE:
Base styles
:hover
Hover styling
:active
Selected styling
:visited
Visited
.cui-button--error
Error styling
.cui-button--error:active
Error selected styling
.cui-button--cancel
Cancel button styling
.cui-button--full-width
Full width of container
.cui-button--small-rounded
Small and rounded
.cui-button--inverted
Inverted
CODE:
<button class="cui-button [modifier class]">Button</button>
