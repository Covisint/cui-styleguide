1.3 Forms

DESCRIPTION:

Styles for form elements

1.3.1 Inputs

EXAMPLE:
Default styling
:focus
Styling when user clicks in a field to enter input
.cui-input--error
Styling after user has entered input that is invalid
.cui-input--valid
Styling after user has entered input that is valid
CODE:
<input type="text" class="cui-input [modifier class] cui-input--full"/>

1.3.2 Dropdown

EXAMPLE:
CODE:
<select class="cui-input cui-input--full cui-select">
            <option value="helloo"></option>
            <option value="hellloooo"></option>
            <option value="hellllooooooo"></option>
            <option value="HELLO"></option>
          </select>
